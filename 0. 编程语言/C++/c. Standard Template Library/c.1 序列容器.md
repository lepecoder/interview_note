## 序列容器

- array
  可以存储N个T类型的元素，不能增删容器长度。
- vector
  在尾部增删效率为O(1)，在其他位置插入删除效率为O(n)
- deque
  在头尾插入删除都是O(1)
- list
  双向链表，在任意位置都能高效的插入删除，但访问很慢
- forward_list
  单链表，只能从第一个元素开始访问
