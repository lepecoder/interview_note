## 模式分类

| 类型       | 描述                                             |
| ---------- | ------------------------------------------------ |
| 创建型模式 | 用于构建对象，以便它们可以从实现系统中分离出来。 |
| 结构性模式 | 用于在许多不同的对象之间形成大型对象结构。       |
| 行为型模式 | 用于管理对象之间的算法、关系和职责。             |
